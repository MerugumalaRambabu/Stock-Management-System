# Stock Management System (Spring Boot + Thymeleaf + JPA)

This is a simple stock/inventory management web application built with Spring Boot, Thymeleaf and JPA (MySQL).

## Features
- Add, edit, delete, and list stock items
- Search and filter by product name
- Low-stock indicator

## Tech
- Java 17
- Spring Boot (Web, Data JPA, Thymeleaf)
- MySQL

## Run
1. Update `src/main/resources/application.properties` with your DB credentials.
2. `mvn spring-boot:run` or run the main class from your IDE.
3. Open `http://localhost:8080/stocks`
